# ຄິດໄລ່ໂພຍ — ວິທີຕິດຕັ້ງເປັນແອັບ (Install as app)

ໄຟລ໌ໃນຊຸດນີ້ / ไฟล์ในชุดนี้

```
LOTTERY-CALCULATOR/
├── index.html              ← ຕົວແອັບ
├── manifest.webmanifest    ← ຂໍ້ມູນແອັບ (ຊື່, ໄອຄອນ, ສີ)
├── sw.js                   ← ໃຊ້ໄດ້ຕອນບໍ່ມີເນັດ (offline)
├── assets/                 ← ໄອຄອນ
├── START-WINDOWS.bat       ← ເປີດເຊີບເວີໃນເຄື່ອງ (Windows)
├── server.ps1              ← ເຊີບເວີສຳຮອງ ໃຊ້ PowerShell (ບໍ່ຕ້ອງລົງຫຍັງ)
└── START-MAC.command       ← ເປີດເຊີບເວີໃນເຄື່ອງ (Mac)
```

---

## ວິທີທີ 1 — ໄວທີ່ສຸດ, ບໍ່ຕ້ອງຕິດຕັ້ງຫຍັງເພີ່ມ (Windows / Mac)

ເໝາະສຳລັບໃຊ້ຄົນດຽວໃນເຄື່ອງຕົນເອງ
*(ไทย: เร็วที่สุด ไม่ต้องลงอะไรเพิ่ม เหมาะกับใช้เครื่องตัวเอง)*

1. ຄລິກຂວາທີ່ `index.html` → Open with → **Google Chrome** (ຫຼື Microsoft Edge)
2. ໃນ Chrome ກົດ **⋮** (ມຸມຂວາເທິງ) → **Cast, save and share** → **Create shortcut…**
   - Edge: **⋯** → **Apps** → **Install this site as an app**
3. ຕັ້ງຊື່ວ່າ `ຄິດໄລ່ໂພຍ` ແລ້ວ **ຕິກ ☑ Open as window**
4. ກົດ **Create**

ຈະໄດ້ໄອຄອນຢູ່ໜ້າ Desktop ເປີດຂຶ້ນມາເປັນໜ້າຕ່າງແອັບ ບໍ່ມີແຖບ URL

> ຂໍ້ຈຳກັດ: ວິທີນີ້ຈະບໍ່ມີປຸ່ມ "ຕິດຕັ້ງແອັບ" ໃນແອັບ ແລະ ບໍ່ມີລະບົບ offline cache
> (ແຕ່ບໍ່ເປັນຫຍັງ ເພາະໄຟລ໌ຢູ່ໃນເຄື່ອງແລ້ວ — ຕ້ອງມີເນັດສະເພາະຄັ້ງທຳອິດເພື່ອໂຫຼດຟອນ)

---

## ວິທີທີ 2 — ຕິດຕັ້ງເປັນ PWA ເຕັມຮູບແບບ (ມີ offline)

ບໍ່ຕ້ອງຕິດຕັ້ງໂປຣແກຣມເພີ່ມ — ຖ້າບໍ່ມີ Python ຫຼື Node
ມັນຈະໃຊ້ PowerShell ທີ່ມີໃນ Windows ຢູ່ແລ້ວ (`server.ps1`)
*(ไทย: ไม่ต้องลง Python ก็ได้ ถ้าไม่มีจะใช้ PowerShell ที่ติดมากับ Windows แทน)*

1. **Windows**: ດັບເບິນຄລິກ `START-WINDOWS.bat`
   **Mac**: ດັບເບິນຄລິກ `START-MAC.command`
2. ເບຣົາເຊີຈະເປີດ `http://localhost:8777/index.html` ໃຫ້ເອງ
3. ກົດປຸ່ມສີຂຽວ **⤓ ຕິດຕັ້ງແອັບ** ຢູ່ແຖບເທິງ (ຫຼື ຮູບ ⊕ ໃນຊ່ອງ URL)
4. ໄດ້ໄອຄອນຢູ່ Desktop / Start Menu ຄືແອັບທົ່ວໄປ

ຫຼັງຕິດຕັ້ງແລ້ວເປີດໃຊ້ໄດ້ເລີຍ **ບໍ່ຕ້ອງເປີດ START ອີກ** ຖ້າບໍ່ໄດ້ລຶບ cache

## ວິທີທີ 3 — ອອນລາຍ ໃຊ້ໄດ້ທັງມືຖື (ແນະນຳຖ້າໃຊ້ຫຼາຍເຄື່ອງ)

*(ไทย: อัปขึ้นเว็บฟรี แล้วติดตั้งลงมือถือได้ด้วย)*

1. ອັບໂຫຼດໂຟນເດີນີ້ຂຶ້ນ **Netlify Drop** (https://app.netlify.com/drop)
   ຫຼື **GitHub Pages** — ບໍ່ເສຍຄ່າ
2. ເປີດລິ້ງທີ່ໄດ້ດ້ວຍ Chrome
3. **ຄອມ**: ກົດ ⤓ ຕິດຕັ້ງແອັບ
   **Android**: ⋮ → Add to Home screen
   **iPhone / iPad**: ປຸ່ມແບ່ງປັນ ⎋ → Add to Home Screen

## ຂໍ້ຄວນຮູ້

- ຂໍ້ມູນໂພຍທັງໝົດເກັບຢູ່ **ໃນເຄື່ອງເທົ່ານັ້ນ** (localStorage) ບໍ່ໄດ້ສົ່ງອອກໄປໃສ
- ຄົນລະວິທີເປີດ = ຄົນລະບ່ອນເກັບ. ຖ້າເປີດຈາກ `file://` ແລ້ວມາເປີດຈາກ
  `localhost` ຈະບໍ່ເຫັນປະຫວັດອັນເກົ່າ — ໃຫ້ເລືອກໃຊ້ວິທີດຽວແຕ່ຕົ້ນ
- ຢ່າລຶບ cache / site data ຂອງເບຣົາເຊີ ບໍ່ດັ່ງນັ້ນປະຫວັດຈະຫາຍ
  (ແນະນຳ: ກົດ **ດາວໂຫຼດ Excel** ເກັບໄວ້ເປັນສຳຮອງ)
- ຕ້ອງມີເນັດຄັ້ງທຳອິດ ເພື່ອໂຫຼດຟອນ Noto Sans Lao — ຫຼັງຈາກນັ້ນໃຊ້ offline ໄດ້

## ຖ້າ START-WINDOWS.bat ຂຶ້ນ error

- ຢ່າແກ້ໄຟລ໌ `.bat` ດ້ວຍໂປຣແກຣມທີ່ບັນທຶກເປັນ UTF-8 —
  ຕົວໜັງສືລາວໃນໄຟລ໌ `.bat` ຈະເຮັດໃຫ້ cmd ອ່ານຄຳສັ່ງຜິດ
  *(ไทย: ห้ามใส่ตัวอักษรลาว/ไทยในไฟล์ .bat — cmd จะอ่านคำสั่งเพี้ยน)*
- ຖ້າຂຶ້ນວ່າ port ຖືກໃຊ້ຢູ່ ໃຫ້ແກ້ `set PORT=8777` ໃນ `START-WINDOWS.bat`
  ແລະ `$port = 8777` ໃນ `server.ps1` ເປັນເລກອື່ນ ເຊັ່ນ 8899
- ຫຼື ໃຊ້ **ວິທີທີ 1** ໄປເລີຍ ບໍ່ຕ້ອງເປີດເຊີບເວີ
